## Sales Performance Dashboard (Excel)
## Overview

This project presents a Sales Performance Dashboard created using Microsoft Excel.
It converts raw sales data into clear, visual insights to help understand business performance.

## Key Highlights

* Cleaned and organized sales data
* Built interactive Pivot Tables & Charts
* Tracked key metrics like Sales and Orders
* Analyzed category and product performance
* Designed an easy-to-read dashboard

## Tools Used

**Microsoft Excel**
(Pivot Tables, Charts, Formulas, Data Visualization)

## What the Dashboard Shows

* Overall sales performance
* Sales trends and comparisons
* Category-wise contribution
* Product-level insights

## Skills Demonstrated

Data Analysis • Data Cleaning • Dashboard Design • Business Insights • Excel
